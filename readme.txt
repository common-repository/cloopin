Cloopin API provides object tagging and sharing on the image/video.

== Description ==

Cloopin is a social media platform which allows users to tag(Cloopin) any specific object within images and videos on any web site and provides real-time communication among users(Cloopiners), such as your families, friends, and colleagues, via tagged objects.
Do simply tags(Cloopin), it automatically store in Cloopin server, so even if you do not want to share or communicate, you can see your collections on Cloopin web site.
Simply sign up with your Facebook, Twitter, and Google+ account or your own email address. 
By doing that, now you can use both a browser extension and a mobile app which can be used on every device such as PCs, Tablets, and Smartphones for Cloopin.

== Installation ==

**Installation**

1. Search keyword for "Cloopin"
2. Install the "Cloopin" Plugin
3. Activate the plugin
4. And you're done !

**Manual Installation**

> *  Insert Three JS file and one CSS file in <head> like below example
`       
                    
        <head> 
            ... 
            <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js" type="text/javascript" charset="utf-8"></script>
            <script src="https://cloopin.com/static/serverapi/cloopForeground.min.js" type="text/javascript" charset="utf-8"></script>
            <script src="https://cloopin.com/static/js/apiCloop.js" type="text/javascript" charset="utf-8"></script>
            <link rel="stylesheet" href="https://cloopin.com/static/serverapi/cloopui.css">
            ...
        </head>
`
